# 效果 Preset（默认免费 · 智能满配选型）

剪映资源库很大：转场、场景特效、人物特效、滤镜、视频入出/组合动画、字体、文字动画。  
**目标：按片段数量与内容尽量用上合适类别**，不是只贴一层边框。

选用前查目录（统一入口）：

```bash
jy-compile catalog --type transitions --free --limit 80
jy-compile catalog --type effects --kind scene --grep 边框 --free
jy-compile catalog --type effects --kind character --free --limit 40
jy-compile catalog --type filters --free --grep 日系
jy-compile catalog --type video-intros --free --grep 渐
jy-compile catalog --type video-outros --free --grep 渐
jy-compile catalog --type video-groups --free --limit 40
jy-compile catalog --type fonts --free --grep Anton
jy-compile catalog --type text-intros --free --limit 40
jy-compile catalog --type text-loops --free --grep 扫光
jy-compile validate /path/to/plan.json   # 错位 → ok:false，禁止强行编译
```

**错位禁止**（会导致草稿崩溃）：边框≠转场≠滤镜≠视频动画≠文字动画≠字体。一名两用时按**用途**放字段。

VIP 素材须先提醒用户确认本机有**剪映 VIP**；未确认一律 `--free`。

**原则：按片段内容选效果；禁止全片同一特效/同一转场/同一滤镜复制粘贴；禁止同段堆叠多个场景/人物特效。**

---

## 同段特效预算（强制）

每个视频片段（同一时间窗）硬上限：

| 类别 | 上限 | 说明 |
|------|------|------|
| 场景特效 `overlays type=effect`（非人物） | **1** | 禁止同段「边框 + 漏光 + 闪烁」叠罗汉 |
| 人物特效 | **1** | 只用 `clips[].character_effect`，不要再叠人物 overlay |
| 滤镜 | **1** | — |
| 视频动画 | **1 套** | `intro`+`outro`，或单独 `group_animation` |
| 蒙版 | **0～1** | 默认 0 |

合法：`滤镜 + 入出动画 + 人物特效 + 1 个场景特效`。  
非法：同段 2 个场景特效；或 `character_effect` + 人物类 effect overlay。`jy-compile validate` 会拦截。

---

## 按片段数量的「满配」强度

| clip 数 | 必须覆盖（每段仍遵守预算） | 建议加码 | 慎用 |
|---------|----------------------------|----------|------|
| 1 | 滤镜 + 视频 intro/outro；有人则 +人物；再选 **1** 个场景特效 | 标题字 + 字体 + 文字 intro | `group_animation`；同段第二特效 |
| 2–3 | 每段不同滤镜；入出或 1 段 group；每段 **≤1** scene；有人段人物特效；接缝转场 | 字幕/标题字体；轻 loop（标题） | 同段叠特效；每段厚边框 |
| 4–6 | 上表 + 转场 ≥2 种；相邻主特效名不同；group 全片最多 1–2 段 | 促销标题 `Anton`/`站酷*` + `扫光` | 全段 group；蒙版遮脸 |
| 7+ | 转场 ≥3 种；特效/滤镜池轮换；group 仅 1 段点睛 | 批量变体换滤镜组+动画组+BGM | 每段 mask / 强故障 |

---

## 内容 → 场景特效（`overlays type=effect`）

每段只选 **1 个**（不要「主特效 + 轻量点缀」叠两层）。

| 片段内容特点 | 优先特效（三选一） | 避免 |
|--------------|-------------------|------|
| 服装/产品静物、细节特写 | `撕纸涂鸦边框` / `纸质撕边` / `镭射边框` | 强烈故障盖产品；同段再加漏光 |
| 全身走秀、大景别、构图稳 | `胶片框 III` / `录制边框 III` / `胶片漏光` | 全程抖动；框+漏光同段双挂 |
| 人物口播、面部近景 | `细闪` / `星光闪闪`（轻）；边框慎用 | 撕纸/强边框遮脸 |
| 快速运镜、动作、切点密 | `胶片抖动` / `定格闪烁` / `故障` | 厚重胶片框拖节奏 |
| 暗调、情绪、氛围 | `胶片漏光` / `细闪` | 高亮镭射边框 |
| 明亮、活泼、促销感 | `镭射边框` / `星光闪闪` / `定格闪烁` | 过重胶片旧感 |

相邻 clip 主特效名应不同；同名间隔至少 1 段。

---

## 内容 → 人物特效（`clips[].character_effect`）

挂在**片段本身**，比整轨 effect 更贴人物。  
**规则：画面里有人就可以挂**——不限于口播；模特试穿、半身/全身产品展示、走秀同理。仅纯静物/无人产品特写才默认不加。

| 内容 | 优先（免费示例） | 避免 |
|------|------------------|------|
| 口播 / 面部近景 | `光环 I`、`光环 II`（短时或全程轻挂） | `BOOM！`、`crash！`、夸张打击类 |
| 模特产品展示 / 试穿（半身·全身） | `光环 I`/`光环 II`；或轻拖尾/氛围类（先 `--grep` 确认免费）；相邻段换不同名 | 遮挡服装/产品主体的厚贴脸、夸张爆炸类 |
| 走秀 / 全身动态 | 同上，可选更动感的人物特效；强度宜轻 | 全程强抖动挡衣服细节 |
| 纯静物 / 无人产品 | **默认不加** character_effect | 强行挂人物特效 |

不确定名称时：`jy-compile catalog --type effects --kind character --free --grep …`

---

## 内容 → 滤镜（`clips[].filter`）

每段尽量选滤镜；相邻段不要完全同名。强度口播/产品建议 **40–65**，胶片感可 **70–90**。

| 气质 | 免费示例 | intensity |
|------|----------|-----------|
| 清新明亮 | `清新`、`日系奶油` | 45–60 |
| 冷感时尚 | `冷白`、`冷蓝`、`清冷` | 40–55 |
| 胶片复古 | `Lofi II`、`VHS III`、`喜庆胶片`、`复古工业` | 60–85 |
| 暖促销 | `暖食`、`清新漫和` | 50–70 |

---

## 内容 → 视频入出 / 组合动画

| 字段 | 何时用 | 免费示例 |
|------|--------|----------|
| `intro` + `outro` | **默认**：多数片段用轻入出，增强节奏 | 入：`渐显`/`轻微放大`/`动感放大`/`向上滑动`；出：`渐隐`/`缩小`/`轻微放大`/`向下滑动` |
| `group_animation` | 片段 ≥3 且该段要「分割/冲击」点睛；与 intro/outro **互斥** | `三分割`、`左右分割`、`上下分割`、`抖入放大`、`放大弹动` |
| 节奏 | 快切片：入出要短（靠素材 trim）；柔片：`渐显`/`渐隐` | 不要每段都 `旋转开幕` |

规则：

- 2 段以上时，**至少一半片段**写视频 intro/outro（或等价 group）
- 全片最多 **1–2 个** `group_animation`，优先首段或高潮段
- **禁止**把文字 `渐显` 写进 `clips[].intro`（文字动画只给 overlays）

---

## 内容 → 转场（`junctions`）

| 接缝两端关系 | 优先转场 | 时长参考 |
|--------------|----------|----------|
| 同色系/同场景顺接 | `色彩溶解`、`竖向模糊` | 400–600ms |
| 景别大跳 / 节奏提速 | `闪白`、`抖动 II` | 300–500ms |
| 风格突变 / 强调冲击 | `故障`、`雪花故障` | 300–450ms |
| 柔和情绪过渡 | `色彩溶解` | 500–700ms |

相邻接缝不要重复同一 `transition`；≥3 段至少 **2 种**转场。

---

## 字体与文字动画（有字幕/标题时）

| 用途 | font | 动画 |
|------|------|------|
| 促销大标题 / 英文 slogan | `Anton`、`Montserrat-Black` | intro `弹入`/`放大`；可选 loop `扫光`；outro `弹出` |
| 中文品牌/卖点 | `抖音美好体`、`站酷酷黑体`、`站酷文艺体` | intro `渐显`/`弹入`；loop 慎用 |
| 口播字幕 | `思源中宋` 或默认即可 | 一般无 loop；最多轻 `渐显` |
| 情绪花字 | 站酷系 | loop `闪烁` 仅短窗 |

禁止默认每条字幕都打字机/故障；`loop` 只给标题类短窗。

---

## 蒙版（`clips[].mask`）

默认**不用**。仅当用户要异形裁切、或创意分屏暗示时：`圆形`/`爱心`/`星形` 等，且确认不挡产品/脸。

---

## 气质候选池（按风格取用，仍须分段）

### fashion-rich（时装 / 多品）

- **转场**：`竖向模糊`、`闪白`、`故障`、`抖动 II`、`色彩溶解`
- **场景特效**：`细闪`、`撕纸涂鸦边框`、`胶片框 III`、`胶片漏光`、`镭射边框`、`胶片抖动`、`纸质撕边`、`星光闪闪`
- **滤镜**：`日系奶油` / `冷白` / `Lofi II` 轮换
- **视频动画**：多数段 `渐显`+`渐隐` 或 `轻微放大`+`缩小`；1 段 `三分割`/`左右分割`
- **人物**：有模特/口播的段尽量挂（如 `光环 I`/`光环 II` 轮换）；纯静物段不加
- **字**：标题 `Anton` 或 `抖音美好体` + `弹入`/`扫光`

### clean-border（干净边框）

- 转场：`色彩溶解`、`竖向模糊`
- 特效：`撕纸涂鸦边框`、`白噪点边框`、`纸质撕边`（分段换）
- 滤镜：`清新`、`冷白`（轻强度）
- 动画：轻 `渐显`/`渐隐`；少用 group

### glitch-fast（快切故障）

- 转场：`故障`、`雪花故障`、`闪白`、`抖动 II`
- 特效：动作段 `胶片抖动`；末段短 `定格闪烁`
- 滤镜：`VHS III`、`Lofi II` 穿插
- 动画：`动感放大`/`动感缩小`；1 段 `抖入放大`

### film-look（胶片）

- 转场：`色彩溶解`、`竖向模糊`
- 特效：部分段框、部分段仅漏光/细闪
- 滤镜：`喜庆胶片`、`复古工业`、`Lofi II`
- 动画：柔和入出；group 少用

---

## 批量变体差异维度（§1.0）

每版至少换 **两类**，优先三类都换：

1. 转场池  
2. 场景特效池 **或** 滤镜组  
3. 视频动画策略（intro/outro 套 vs 1 个 group）**或** BGM mood  

---

## 安全默认（意图不清时）

仍须**分段不同**，且尽量写满「滤镜 + 视频入出 + 分段特效 + 转场」：

- 2–3 段：每段不同滤镜 + 不同 scene effect；接缝换转场；视频 intro/outro 至少 2 段有
- 反例（禁止）：全程一个边框 + 全接缝同一转场 + 无滤镜 + 无视频动画
