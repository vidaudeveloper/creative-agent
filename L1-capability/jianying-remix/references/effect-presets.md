# 效果 Preset（默认免费 · 按内容选型）

下列名称均应为目录内 `is_vip=false` 项。选用前可用：

```bash
jy-compile effects --grep <关键词>
jy-compile transitions --limit 100
jy-compile text-animations --kind intro --free --limit 40
jy-compile text-animations --kind outro --free --grep 渐
```

若用户允许 VIP，可另选 `撕纸拼贴`、`快速缩放`、`复古撕纸` 等（以本机目录 `is_vip` 为准）。**选用前必须提醒用户**：VIP 素材需要本机剪映登录有效 **剪映 VIP 账号**；请用户确认有 VIP 后再写入 Plan，否则坚持免费项。

**原则：按片段内容选效果，禁止全片同一特效/同一转场。** Preset 名只表示整体气质与候选池，不是「整条时间轴套一套」。

## 文字高亮与动画（按需）

| 用途 | 推荐 |
|------|------|
| 卖点/品牌词强调 | `keywords` + `keyword_color` `#ff7100`（或品牌色） |
| 标题入场 | `渐显` / `弹入` / `放大`（免费） |
| 标题出场 | `渐隐` / `弹出` / `缩小` |
| 口播字幕 | 一般不加动画；需要时只用轻 `渐显` |

禁止默认给每条字幕都加打字机/故障类动画。
## 内容 → 特效（分段）

| 片段内容特点 | 优先特效（选 1，必要时 +1 轻量） | 避免 |
|--------------|----------------------------------|------|
| 服装/产品静物、细节特写 | `撕纸涂鸦边框`、`纸质撕边`、`镭射边框` | 强烈故障盖脸 |
| 全身走秀、大景别、构图稳 | `胶片框 III`+短时 `胶片漏光`、`录制边框 III` | 全程抖动 |
| 人物口播、面部近景 | 轻量 `细闪` 或短时 `星光闪闪`；边框慎用或极淡 | 撕纸/强边框遮脸 |
| 快速运镜、动作、切点密 | `胶片抖动`、短时 `定格闪烁`、`故障`（短） | 厚重胶片框拖节奏 |
| 暗调、情绪、氛围 | `胶片漏光`、`细闪` | 高亮镭射边框 |
| 明亮、活泼、促销感 | `镭射边框`、`星光闪闪`、`定格闪烁` | 过重胶片旧感 |

同一成片内：相邻 clip 的主特效名应不同；同名特效间隔至少 1 段。

## 内容 → 转场（按接缝）

| 接缝两端关系 | 优先转场 | 时长参考 |
|--------------|----------|----------|
| 同色系/同场景顺接 | `色彩溶解`、`竖向模糊` | 400–600ms |
| 景别大跳 / 节奏提速 | `闪白`、`抖动 II` | 300–500ms |
| 风格突变 / 强调冲击 | `故障`、`雪花故障` | 300–450ms |
| 柔和情绪过渡 | `色彩溶解` | 500–700ms |

相邻 `junctions` 不要重复同一 `transition`；整片至少用 **2 种以上** 转场（仅 2 段素材时除外，可 1 种）。

## 气质候选池（按风格取用，仍须分段）

### fashion-rich（时装 / 多品）

- **转场池**：`竖向模糊`、`闪白`、`故障`、`抖动 II`、`色彩溶解`（按接缝轮换，勿全用一种）
- **特效池**：`细闪`（可作极短点缀）、`撕纸涂鸦边框`、`胶片框 III`、`胶片漏光`、`镭射边框`、`胶片抖动`、`纸质撕边`、`星光闪闪`、`录制边框 III`、`定格闪烁`
- 每段 trim 约 2.5–3.5s；**每段从池中按内容表选不同组合**

### clean-border（干净边框）

- 转场池：`色彩溶解`、`竖向模糊`（接缝轮换）
- 特效池：`撕纸涂鸦边框`、`白噪点边框`、`纸质撕边`（分段换，勿全程同一边框）

### glitch-fast（快切故障）

- 转场池：`故障`、`雪花故障`、`闪白`、`抖动 II`
- 特效：仅在动作段用 `胶片抖动`；末段可短时 `定格闪烁`；静物/口播段改轻特效或不用故障

### film-look（胶片）

- 转场池：`色彩溶解`、`竖向模糊`
- 特效：部分段 `胶片框 III`，部分段仅 `胶片漏光` 或 `细闪`，避免五段全是同一胶片框

## 安全默认（意图不清时）

仍须**分段不同**：

- 2–3 段以上：每段各选不同边框/轻效；接缝至少换 2 种转场
- 反例（禁止）：全程一个 `撕纸涂鸦边框` + 所有接缝都是 `竖向模糊`
