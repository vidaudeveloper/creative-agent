# Camera & Shot Vocabulary

Use **one dominant camera move per beat**. Mixing conflicting moves degrades output.

## Basic moves

| Term | Use when |
|------|----------|
| 推镜头 / slow push in | Reveal product, intensify emotion |
| 拉镜头 / pull back | Establish context, ending |
| 左摇 / 右摇 | Scan environment, follow action |
| 跟拍 / tracking | Subject walks, UGC follow |
| 环绕 / orbit | Product hero, 360 feel |
| 固定镜头 / static | Product spin, typography hold |
| 一镜到底 | Continuous scene, no cuts |

## Advanced

| Term | Use when |
|------|----------|
| 希区柯克变焦 | Shock, tension beat |
| 低角度仰拍 | Hero product, power |
| 俯拍 / 鸟瞰 | Layout, unboxing top-down |
| 第一人称 POV | Hands-on demo |
| 机械臂跟随 | Complex action, game-like |

## Framing

| Term | Frame |
|------|-------|
| 大特写 | Eyes, logo, product detail |
| 特写 | Face (stylized only) or product fill |
| 中近景 | Shoulders up / product + hands |
| 中景 | Waist up |
| 全景 | Full body / full product + scene |
| 远景 / establishing | Environment first |

## Storyboard shorthand (for Final Video Spec)

When writing script2film scenes, append camera hints:

```
Shot 3 (6s): Medium close-up, slow push-in, warm window light —
hands unbox @product, diegetic tear sound, no BGM.
```

## VidAU aspect ratios

| Ratio | Typical use |
|-------|-------------|
| 9:16 | TikTok, Reels, Shorts |
| 16:9 | YouTube, Amazon hero |
| 1:1 | Feed square |

State ratio in strategy block; MCP uses `aspect_ratio` param separately from prompt text.
