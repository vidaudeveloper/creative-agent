# 场景规划原则（按产品与需求动态决定）

**不要写死场景。** 每段意图必须根据：

1. **产品图特点**（品类、版型/配色、材质、使用方式、目标气质）  
2. **用户需求**（受众、渠道、季节、风格、是否要模特、禁忌等）

现场推导 N 个互不重复的展示意图；下表与示例只是**灵感维度**，禁止原样照抄当固定剧本。

每段最终都要经 **creative-seedance2-prompt** 写成完整视频 prompt。

## 规划流程（每次必做）

1. 看图：产品是什么、卖点外观、适合什么场合穿着/使用  
2. 听需求：用户有没有指定风格、人群、场景、段数  
3. 定 `clip_count`（默认 5，可按用户改）  
4. 为每段写自定义 `label` + 一句意图（贴合该 SKU，不要套通用模板句）  
5. 把规划表给用户确认（可改）；未反对再生成

## 服饰 / 鞋子 / 服装配饰

识别为服装、鞋子、箱包、配饰等时：

- **默认** `clip_count=5`
- **默认** 以真人穿着/穿戴展示为主（不要默认整段平铺静物）
- **默认** **不同模特 × 不同场景**（五段换脸、换地点/光线/背景）
- **具体哪 5 个场景、什么模特气质**：完全由产品特点 + 用户需求决定  
  - 例：冲锋衣 → 可能偏户外徒步、雨天城市、营地等（仍须彼此不同）  
  - 例：礼服 → 可能偏晚宴、红毯感影棚、宴会厅等  
  - 例：跑鞋 → 可能偏跑道、街头慢跑、健身房等  
  - 用户若只要「办公室通勤三镜」→ 服从用户，不要强行塞街拍/影棚

**硬约束（场景内容可变，规则不变）：**

- 相邻段模特外貌气质至少一项明显不同  
- 相邻段场景地点/光线/背景明显不同  
- 单品外观锁定参考图（颜色、版型、鞋型、材质纹理）  
- 4 秒一个清晰动作；面料特写最多做段内短推近，勿占满一整段静物（除非用户只要静物）

## 其它品类（同样动态）

美妆、食品、数码、家居等：**同样先看图 + 需求再定场景**，不要套固定五段清单。

可参考的**维度**（选用，非必选顺序）：

| 维度 | 何时考虑 |
|------|----------|
| 英雄展示 | 需要第一眼认品 |
| 使用/涂抹/开封动作 | 产品强依赖「怎么用」 |
| 生活/职场/户外环境 | 需要场景共鸣 |
| 材质/质地特写 | 卖点在工艺或口感视觉 |
| 结果/收束 | 需要闭环或种草结尾 |

选哪些维度、怎么组合，以**该产品 + 用户这句话**为准。

## 写作约束（Seedance）

- 产品外观锁定参考图  
- 相邻段构图、镜头运动、场景意图都要有差异  
- 服饰类额外保证换模特  
- 4 秒内不堆剧情  
- 竖屏 9:16；主体勿贴边被裁
