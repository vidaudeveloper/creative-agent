# Script templates (handheld talking-head)

Adapted from common UGC/avatar pacing practice. Each beat’s **on-screen action** = hold / tilt / point at product **while speaking**.

## Default visual (required)

**全程人物手持产品口播** — Hook→CTA 每一镜都是 talking-handheld + lipsync。  
**禁止**默认规划：产品静物特写、开箱 B-roll、无人脸的展示镜。  
仅当用户明确说要「产品展示 / 特写 / 开箱」时，才加对应镜。

## Pacing (spoken)

| Tone | EN words/sec | EN words / 30s | ZH chars/sec (approx) | ZH chars / 30s |
|------|--------------|----------------|------------------------|----------------|
| Conversational | 2.5–3.0 | 75–90 | ~3.5–4.5 | ~105–135 |
| Professional | 2.0–2.5 | 60–75 | ~3.0–3.8 | ~90–115 |
| Energetic / sales | 3.0–3.5 | 90–105 | ~4.5–5.5 | ~135–165 |

Per Seedance shot: keep spoken length so TTS lands in **4–12s** after `+0.4s` tail.

## Authenticity markers

- Contractions: "I'm" / "don't" (ZH: 口语短句、语气词 OK)
- Fragments: "Pretty cool, right?"
- Direct address: "You're gonna love this."
- Self-correction: "It's fast — actually, really fast."
- Phonetic hints for brand names in TTS text if needed

## 15s template (2 shots typical)

```
HOOK (0–3s): Pattern interrupt / question — 8–12 EN words (~12–18 ZH chars)
BRIDGE (3–7s): Hook → product in hand — 15–20 EN words
BENEFIT (7–12s): Core value while showing product — 15–20 EN words
CTA (12–15s): Clear next step — 8–12 EN words
```

**Visual (default)**: Shot1 + Shot2 both mid handheld talking; product stays in hand.

## 30s template (3–5 shots)

```
HOOK (0–3s): Attention — 8–12 EN words
PROBLEM (3–8s): Relatable pain — 15–25 EN words
SOLUTION (8–15s): Product in hand + key feature — 20–30 EN words
PROOF (15–22s): Demo / social proof while still talking handheld — 15–25 EN words
CTA (22–30s): Urgency + next step — 15–25 EN words
```

**Visual (default)**: All shots talking-handheld. Vary gesture (tilt / point) — **not** cutaway to product-only.

## 60s template (5 shots)

```
HOOK (0–5s)
STORY/PROBLEM (5–15s)
DISCOVERY (15–25s) — product already in hand while speaking
FEATURES (25–40s) — 2–3 benefits, still talking handheld
PROOF (40–50s)
CTA (50–60s)
```

## Do / Don't

| Do | Don't |
|----|-------|
| Short sentences (8–15 EN words) | Long brochure paragraphs |
| Pause with "..." for TTS | Cram features into one breath |
| Name what the hand is doing | Face-only with product off-screen |
| Keep face + product every default shot | Default product macro / B-roll “for variety” |
| End on a clear action | Fake reviews / invented SKUs |
| Add showcase only if user asks | Assume user wants still-life inserts |
