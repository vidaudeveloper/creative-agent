# Seedance Prompt Templates

Reusable patterns for VidAU ad and short-form workflows.

## Product ad (15s, segmented)

```
参考 @产品图 的外观，制作一段15秒竖屏产品广告。
0–3秒：产品从暗场中旋转入场，特写材质与 logo 区域，慢推镜头。
4–8秒：多角度切换——正面、侧面、使用场景——产品表面扫光。
9–12秒：生活场景中的实际使用，中景跟拍，自然窗光。
13–15秒：产品主视觉定格，品牌口号位置留白，环境音收束。
音效：产品交互声、轻环境音；无背景音乐。
```

## UGC / TikTok hook (5–8s)

```
竖屏9:16，手持轻微晃动的手机拍摄质感，@产品图 置于画面中心，
快速推近至产品特写，背景虚化，自然日光，真实生活厨房/桌面场景，
无品牌侵权元素，无真人正脸。
音效：开箱撕膜声、产品放置声。
```

## Problem → solution beat (single shot 6s)

```
中近景，主角（无真实人脸，背影或手部）展示痛点场景，
切至 @产品图 产品入画，慢推，产品成为视觉焦点，
暖色治愈氛围，浅景深。
```

## Reference-image-to-video (VidAU default)

```
以 @首帧图 为首帧，保持产品与 @参考图 一致的包装细节与配色，
镜头缓慢环绕产品15度，背景简洁渐变棚拍光，
产品表面高光清晰，无额外道具抢镜。
音效：轻微环境白噪，无音乐。
```

## First/last frame transition

```
从首帧构图开始，镜头平稳向前推进，
主体完成 [具体动作：打开盖子 / 倾倒 / 穿戴]，
运动连贯无跳切，末帧与尾帧构图对齐，
光影与首帧一致。
```

## E-commerce 360 spin

```
@产品图 中的产品悬浮居中，静止镜头，
产品匀速旋转360度两圈后停住，
3D 产品展示特效，干净背景，商业摄影光。
```

## Short drama dialogue (15s)

```
画面（0–5秒）：特写情绪，[具体表情与动作，无 IP 名]。
台词1（角色A，[语气]）："[逐字对白]"
画面（6–10秒）：[反应镜头 + 道具细节]。
台词2（角色B，[语气]）："[逐字对白]"
画面（11–15秒）：[反转或证据揭示]。
音效：对白清晰，环境音克制，无背景音乐。
```

## Compliance rewrite (when blocked)

Replace:

- Brand / character names → "generic hero / premium wireless earbuds / stylized mascot"
- Celebrity likeness → "original character, no resemblance to real person"
- Logo-heavy refs → "minimal unbranded packaging, solid color label"
